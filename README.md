# xDM

> c# 使用大漠插件. 完成 类似 按键精灵的 功能. 方法 注释 正在慢慢的 完善中

# 仓库
## github
https://github.com/xxxxue/xDM
## 码云
https://gitee.com/xxxxue/xDM/tree/dev

# 准备
1. 运行需要 .NET Framework 4.7.2  或者 手动把 类库框架版本改为 自己有的版本
